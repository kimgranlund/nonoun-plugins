# product-forge v0.2 — critic source dossiers (moved out of the repo)

This dossier assembled the verifiable real-world identity, framework, and verbatim-quote sourcing for the council critics — all named, living practitioners. Because it is, by construction, a catalog of real names, companies, books, and source URLs, it was **moved out of the tracked tree** when the critic identities were obscured (v0.3.5), so a search for a practitioner's name or company does not surface this repository.

The full dossier now lives, verbatim, in the git-ignored `agents/.name-map.md` (the same private map that decodes the obscured `critic-<first>-<initial>` slugs / `First L.` display names back to the real practitioners and holds the lifted bios). It is kept out of git by design.

The committed council keeps only the operating **lens** under obscured `First L.` display names; the anti-fabrication sourcing discipline it documented is unchanged and is enforced in CI by `bin/check-sourcing.py`.
